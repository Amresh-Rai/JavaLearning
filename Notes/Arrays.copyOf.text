# 🔥 1. Sabse simple definition

👉 **`Arrays.copyOf()` ka use ek array ki copy banane ke liye hota hai.**

**Short me:**

> Ye method ek **naya array create karta hai** aur usme old array ke elements copy kar deta hai.

---

# 🧠 2. Syntax

```java
Arrays.copyOf(originalArray, newLength);
```

### 🔹 Parameters:

* `originalArray` → jis array ki copy banani hai
* `newLength` → new array ka size

---

# ⚡ 3. Basic Example

```java
import java.util.Arrays;

public class Main {
    public static void main(String[] args) {

        int[] arr = {1, 2, 3};

        int[] copy = Arrays.copyOf(arr, 3);

        System.out.println(Arrays.toString(copy));
    }
}
```

### ✅ Output:

```
[1, 2, 3]
```

---

# 🔥 4. Important case (size change 🔥)

## ✅ Case 1: New size **bada hai**

```java
int[] arr = {1, 2, 3};

int[] copy = Arrays.copyOf(arr, 5);

System.out.println(Arrays.toString(copy));
```

### Output:

```
[1, 2, 3, 0, 0]
```

👉 Extra space me **default value (0)** fill ho jati hai

---

## ✅ Case 2: New size **chhota hai**

```java
int[] arr = {1, 2, 3, 4, 5};

int[] copy = Arrays.copyOf(arr, 3);

System.out.println(Arrays.toString(copy));
```

### Output:

```
[1, 2, 3]
```

👉 Baaki elements cut ho jate hain ✂️

---

# 💡 5. Real Use (VERY IMPORTANT 🔥)

## 🔴 Problem:

```java
int[] arr = {1, 2, 3};
int[] copy = arr;   // ❌ galat
```

👉 Ye copy nahi hai
👉 Dono same memory point kar rahe hain

---

## 😱 Proof:

```java
copy[0] = 100;

System.out.println(Arrays.toString(arr));
```

### Output:

```
[100, 2, 3]
```

---

## ✅ Correct way:

```java
int[] copy = Arrays.copyOf(arr, arr.length);
```

👉 Ab **separate memory** milegi
👉 original safe rahega

---

# 🧠 6. Deep understanding

👉 `Arrays.copyOf()`:

* new array banata hai
* elements copy karta hai
* memory alag hoti hai

---

# ⚠️ 7. Important point (interview trap 🔥)

👉 Primitive array (int, char) → **proper copy**
👉 Object array → **shallow copy**

Matlab:

* objects ke reference copy hote hain, object khud nahi

---

# 🚀 8. Short summary (yaad rakhne wala)

👉 `Arrays.copyOf()` =
✔ new array banata hai
✔ elements copy karta hai
✔ size change bhi kar sakte ho
✔ safe copying (reference sharing nahi hota for primitives)
